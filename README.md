## Set up

1. check your Python version, and upgrade to v3.3 or over
```sh
$ python3 --version
```

2. set venv for your shell
```sh
$ cd myFirstOpenJij
$ python3 -m venv venv
$ source venv/bin/activate
# see https://docs.python.org/ja/3/library/venv.html#how-venvs-work
```

3. install packages
```sh
$ pip install -r venv/pipfreeze
```
